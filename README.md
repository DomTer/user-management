# user-management

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development -> http://localhost:8080/
```
yarn serve
```

### Generate the schema once
```
yarn apollo:schema:generate
```

### Starts the Apollo Service 
```
yarn apollo:start
```
